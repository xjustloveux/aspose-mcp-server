MCP 辦公文檔處理服務器

為 MCP 客戶端提供強大的辦公文檔處理能力

📄 118 個統一工具

Word(28)、Excel(32)、PowerPoint(26)、PDF(19)、OCR(2)、Email(6)、BarCode(2)、轉換(1)、Session(1)、擴充功能(1) 已整合，提供完整的文檔處理能力。

⚙️ 按需啟用

只啟用需要的文檔類型，減少資源佔用。支援單一組件或全部組件授權。

🌍 跨平台支援

Windows、Linux、macOS (Intel + ARM) 全平台原生支援，無需安裝 libgdiplus 等外部圖形庫，單一可執行檔案，預編譯版本無需安裝 .NET Runtime。

🔒 安全加固

全面的路徑驗證、輸入驗證和錯誤處理，確保系統安全穩定運行。

📋 完整讀寫

支援從A文檔讀取格式應用到B文檔，實現文檔格式的完整遷移和複製。

🔄 跨格式轉換

支援 Word、Excel、PowerPoint、PDF 之間的跨格式轉換，滿足各種文檔處理需求。

🎯 MCP 2025-11-25 規範

完全符合最新 MCP 協議規範，自動工具註解基於命名約定，完整的 JSON-RPC 2.0 錯誤處理。

🔧 自動工具發現

基於命名約定的自動工具註冊系統，無需手動配置即可發現和使用所有工具。

🧪 完整測試套件

780+ 個測試類，10,000+ 個測試用例，使用 xUnit 測試框架，CI/CD 自動測試。

📝 Word (27個工具)

  • 檔案操作（創建、轉換、合併、拆分）
  • 內容編輯（文字、段落、表格、圖片）
  • 格式設定（樣式、頁首頁尾、頁面設定）
  • 高級功能（書籤、超連結、註釋、目錄）

📊 Excel (31個工具)

  • 檔案操作（創建、轉換、合併、拆分）
  • 工作表操作（添加、刪除、重新命名）
  • 資料操作（排序、篩選、資料驗證）
  • 高級功能（圖表、公式、資料透視表）

📽️ PowerPoint (25個工具)

  • 檔案操作（創建、轉換、合併、拆分）
  • 投影片管理（添加、刪除、移動、複製）
  • 內容編輯（文字、圖片、表格、圖表）
  • 高級功能（動畫、轉場、媒體、SmartArt）

📑 PDF (19個工具)

  • 檔案操作（創建、合併、拆分、壓縮、加密）
  • 內容添加（文字、圖片、表格、水印）
  • 編輯操作（編輯文字、表格、書籤、註釋）
  • 高級功能（簽名、頁面旋轉、編輯）

🔍 OCR (2個工具)

  • 圖片文字辨識（PNG、JPG、BMP、TIFF）
  • 掃描 PDF 轉換為可編輯格式
  • 多語言支援與文字定位
  • 前處理功能（歪斜校正、雜訊去除）

📧 Email (6個工具)

  • 郵件格式轉換（MSG、EML 互轉）
  • 郵件屬性提取（主題、寄件者、收件者）
  • 附件管理（提取、列出、刪除附件）
  • 完整保留原始郵件內容與格式

📊 BarCode (2個工具)

  • 條碼生成（1D、2D、郵政條碼）
  • 條碼辨識（從圖片讀取條碼）
  • 支援 60+ 種條碼格式
  • QR Code、Data Matrix、PDF417 等

🔌 擴充功能 (1個工具)

  • 外部程序訂閱 Session 變更
  • 多種傳輸模式（mmap、stdin、file）
  • 支援 Node.js、Python、.NET、自訂解釋器
  • 即時預覽、同步、合規檢查
