🔍 OCR 文字辨識 (2 個工具)

影像前處理 (1)

  • ocr_preprocessing- 影像前處理以提升辨識品質（6個操作：auto_skew, denoise, contrast, scale, invert, dewarp）
    自動校正傾斜、降噪、對比度調整、縮放、反色、去彎曲等預處理操作

文字辨識 (1)

  • ocr_recognition- 圖片/PDF 文字辨識，含專用辨識模式（5個操作：recognize, recognize_pdf, recognize_receipt, recognize_id, recognize_passport）
    recognize：辨識圖片中的文字（PNG/JPG/BMP/TIFF/PDF），回傳文字、信心度、座標；includeWords=true 可取得每個單詞的 Bounding Box 座標（X, Y, Width, Height）
    recognize_pdf：掃描 PDF 轉可編輯文件（DOCX/XLSX/PDF/TXT），支援 language 參數指定辨識語言
    recognize_receipt / recognize_id / recognize_passport：針對收據、身分證、護照的專用辨識，回傳結構化欄位資料
💡 OCR 說明

OCR 功能預設啟用。如需僅啟用 OCR，可使用 --ocr 參數或 ASPOSE_TOOLS=ocr 環境變數。支援多種語言（英文、中文、德文、法文等）。不支援 Linux ARM64 平台；macOS ARM64 需安裝 Rosetta 2。

save 儲存 Session 變更到檔案（可選 outputPath 參數另存為）
close 關閉 Session（可選擇儲存或捨棄變更）
list 列出所有活動 Session
status 取得特定 Session 狀態

暫存檔操作

操作 說明
list_temp 列出可恢復的暫存檔
recover 恢復暫存檔到原始路徑或指定路徑
delete_temp 刪除特定暫存檔
cleanup 清理過期暫存檔
temp_stats 取得暫存檔統計資訊
💡 Session 說明

Session 功能讓您在多次操作之間保持文檔狀態，避免重複讀取和儲存。當 Session 因閒置逾時而結束時，變更會儲存到暫存檔，可透過暫存檔操作恢復。需要透過 --session-enabled 參數或 ASPOSE_SESSION_ENABLED=true 環境變數啟用。

🔌 擴充功能 (1 個)

擴展操作

  • extension- 管理外部擴展程序訂閱 Session 變更（需要啟用 --session-enabled 和 --extension-enabled）（7個操作：list, bind, unbind, status, set_format, bindings, command）

操作說明

操作 說明
list 列出所有已註冊的擴展
bind 將 Session 綁定到擴展，開始接收變更通知
unbind 解除 Session 與擴展的綁定
status 查詢擴展狀態（Running、Stopped、Error 等）
set_format 設定快照輸出格式（PDF、HTML、PNG 等）
bindings 列出 Session 的所有擴展綁定
command 向綁定的擴展發送命令（如 highlight、goto）

💡 擴充功能說明

擴充功能讓外部程序訂閱 Session 文件變更事件，開發者可實現預覽、整合、自動化處理等應用。需要透過 --extension-enabled 參數或 ASPOSE_EXTENSION_ENABLED=true 環境變數啟用，並需要 Session 功能。詳細開發指南請參考 擴充功能開發指南。

⚠️ 評估模式限制

在無授權（評估模式）下運行時，Aspose 組件會有以下限制：

組件 限制說明
Aspose.Words 字型格式（名稱、大小、顏色）可能無法正確應用；文檔會包含評估版水印
Aspose.Cells 多工作表操作受限（如跨工作表存取）；工作簿會包含評估版工作表
Aspose.Slides 文字替換時可能包含水印文字；投影片可能包含評估版標記
Aspose.PDF 集合限制為最多 4 個元素（如頁面數限制）；PDF 會包含評估版水印
Aspose.OCR 辨識結果可能包含限制；部分功能在評估模式下受限

注意事項：

  • 大部分基本操作（讀取、創建、簡單編輯）在評估模式下仍可正常運行
  • 評估模式適合測試和開發，但不建議用於生產環境
  • 使用 test.ps1 -SkipLicense 可在評估模式下運行單元測試
